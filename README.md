# Job Application Tracker

## Projektbeskrivning
Detta är en konsolapplikation i C# som hjälper användaren att hålla koll på sina jobbansökningar.  
Programmet låter dig:
- Registrera nya jobbansökningar
- Visa alla ansökningar
- Filtrera ansökningar efter status
- Sortera ansökningar efter datum
- Uppdatera status på ansökningar
- Ta bort ansökningar
- Visa statistik (totalt antal ansökningar, antal per status, genomsnittlig svarstid)

Projektet använder objektorienterad programmering med klasserna `JobApplication` och `JobManager`, samt LINQ för filtrering, sortering och beräkningar.  

Färgkodning används för status:
- Blå = Applied  
- Gul = Interview  
- Grön = Offer  
- Röd = Rejected
